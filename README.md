jsonapi
=======

A media type for APIs that serve JSON.

More details coming to this README soon, but you can read the spec at

[http://jsonapi.org](http://jsonapi.org)

To build
--------

Make your changes in `source/`, then:

```
$ bundle
$ rake build
$ open build/index.html
```

To deploy:

```
$ git commit
$ git push origin
$ rake publish
```
